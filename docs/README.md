# Project Documentation

Этот каталог содержит рабочую документацию по проекту `WORKOUTSHEDULE`.

## Содержание

- `ARCHITECTURE.md` — общая архитектура и потоки данных.
- `API.md` — backend endpoints, форматы запросов и ответов.
- `DATA_MODEL.md` — модель пользователя, структура данных и каталоги.
- `ML_ADAPTATION.md` — как работает текущая rule-based "ML" адаптация.
- `BACKEND.md` — backend модули, слои, зависимости.
- `FRONTEND.md` — frontend структура, страницы и ключевые состояния.
- `CODE_COMMENTING_POLICY.md` — как документировать код без перегруза.

## Быстрый старт для нового разработчика

1. Прочитать `ARCHITECTURE.md`.
2. Прочитать `API.md`.
3. Прочитать `DATA_MODEL.md`.
4. Для логики подбора и адаптации: `ML_ADAPTATION.md`.
